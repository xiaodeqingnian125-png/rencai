# 批量导入文件选择失败修复

日期：2026-07-24

## 问题原因

- 公寓和户型批量导入使用 `wx.chooseMessageFile`，该接口从微信客户端会话中选择文件，不直接读取手机文件目录或 WPS 本地目录。
- 入口此前没有说明文件来源，且调用时通过 `extension: ["xlsx"]` 预先过滤文件；部分运行环境在文件选择阶段失败后，只显示统一的“选择文件失败”，真实 `errMsg` 被隐藏。
- 现有自动化测试只模拟了文件选择成功，没有覆盖取消、选择器失败和错误文件格式。

## 本次修改

- 点击“批量导入”后先说明：需要将 `.xlsx` 文件发送到文件传输助手或任意微信聊天，再从聊天中选择。
- 调用 `wx.chooseMessageFile` 时不再传入 `extension`，避免选择器扩展名过滤的环境兼容问题。
- 文件选择成功后在业务代码中校验文件名，仅允许 `.xlsx`，其他文件不会上传。
- 用户取消选择时保持静默，并兼容不同大小写的 `cancel` 错误文本。
- 选择器真实失败时记录完整错误，并在弹窗中展示原始 `errMsg` 和恢复操作指引。
- 微信版本不支持文件选择接口时，改为展示升级和操作说明。

## 测试覆盖

- XLSX 正常选择、上传、创建导入任务并进入预览。
- 入口展示文件传输助手操作说明。
- 非 XLSX 文件在上传前被拒绝。
- 用户取消文件选择不显示错误。
- 文件选择器失败时保留真实错误信息和恢复指引。

## 真机验收路径

1. 将编辑后的 `.xlsx` 文件发送到微信文件传输助手。
2. 进入公寓管理或户型管理，点击“批量导入”。
3. 阅读提示后点击“选择文件”，从文件传输助手中选择该文件。
4. 确认进入导入预览页，再执行确认导入。
