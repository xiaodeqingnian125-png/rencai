# 管理员云数据适配器

日期：2026-07-20

## 本次改动

- 新增公寓与户型的管理员表单/云数据库双向字段适配器。
- 公寓展示字段 `rent`、`rooms` 与云端规范字段 `price_min`、`price_max`、`room_summary` 分离。
- 户型展示字段 `apartment`、`rent`、`rooms` 与云端规范字段 `apartment_code`、`price`、`layout` 分离。
- 保存时只输出公寓、户型允许写入的规范字段，避免把 UI 派生字段写入云数据库。
- 平面图字段在适配时过滤掉名称或图片不完整的记录。
- 管理员页仅引入适配器，本批暂不切换现有 CRUD 调用。

## 测试过程

- RED：首次执行适配器测试，因模块不存在按预期失败。
- GREEN：完成最小实现后，公寓读取、公寓保存、户型保存 3 条适配器测试通过。
- 回归：测试基线合计 4 条测试通过；适配器与管理员页通过 `node --check`。
