# 管理员旧图片路径修复

日期：2026-07-20

## 问题

云数据库中旧种子数据的封面字段使用 `apt-1.jpg`、`room-1.jpg` 一类裸文件名。管理员数据接口返回原始值，图片上传组件会将其解析为组件目录下的相对路径，导致微信开发者工具请求本地资源时返回 500。

## 处理

- 管理员数据适配器仅将 `cloud://` 和 `https://` 图片路径传给上传组件。
- 旧裸文件名统一显示为空封面，管理员可通过现有上传控件重新上传并保存。
- 新增回归测试，覆盖公寓和户型的旧文件名清洗。

## 验证

- `node --test tests/admin-adapter.test.js tests/image-uploader.test.js`：6/6 通过。
- 微信开发者工具重新编译：0 errors；保留的 warnings 为基础库和组件 WXSS 兼容性提示。
