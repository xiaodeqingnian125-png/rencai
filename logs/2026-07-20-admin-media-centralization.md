# 管理入口集中化与图片上传保护

日期：2026-07-20

## 本次改动

- 从公寓详情页移除管理员封面图管理、经纬度修正区域及对应写入处理函数。
- 从户型详情页移除管理员户型图管理区域及对应写入处理函数。
- 两个公开详情页移除 `image-uploader` 组件注册，保持纯用户侧展示。
- 图片上传组件增加 5MB 文件大小限制。
- 云存储上传失败时恢复原图片，只触发 `uploaderror`，不再把临时文件路径通过 `change` 当作已保存数据。
- 上传成功和清空图片时同步维护组件内部的上一张图片状态。

## 测试过程

- RED：两个公开详情页仍包含管理员组件和写入函数；云上传失败会覆盖旧图并触发 `change`；超大图片仍进入上传流程。
- GREEN：4 条静态与组件行为测试全部通过。
- 静态检查：图片上传组件、公寓详情、户型详情 JavaScript 语法通过；两个页面 JSON 配置可正常解析。

## 用户体验

管理员只能在管理员管理页面维护封面和位置。用户详情页仍可通过原有“地图”入口查看位置，但不会出现任何管理控件。
