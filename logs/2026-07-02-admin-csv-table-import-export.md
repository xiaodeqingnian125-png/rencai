# 2026-07-02 管理员表格文件导入导出

## 改动范围

- 公寓管理、户型管理的批量导出改为表格格式：
  - 生成 Excel/WPS 可打开的 CSV 内容。
  - 支持在小程序环境中写入临时 CSV 文件并尝试分享文件。
  - 不支持文件分享时，兜底复制 TSV 表格内容到剪贴板。
- 批量导入改为优先上传文件：
  - 支持 `.csv`、`.tsv`、`.txt`。
  - 保留粘贴表格内容作为不支持文件选择时的兜底。
  - 按 `ID` 合并更新；没有匹配 ID 时新增。
- 表格导入支持中文表头，例如：
  - 公寓：`ID`、`公寓名称`、`区域`、`地址`、`租金范围`、`居室类型`、`配套摘要`、`状态`
  - 户型：`ID`、`所属公寓`、`户型名称`、`面积`、`朝向`、`居室`、`租金`、`楼层`、`状态`

## Excel 说明

- 当前原生静态 MVP 不引入第三方 Excel 解析库，因此不直接解析 `.xls/.xlsx` 二进制文件。
- Excel/WPS 可直接打开导出的 CSV；编辑后另存为 CSV（UTF-8）再上传导入。

## 验证

- 已运行全量 JS 语法检查：`rg --files -g '*.js' miniprogram cloudfunctions | xargs -n1 node --check`。
- 已用 Node 页面模拟验证：
  - CSV 公寓表导入后，首页数据可读取。
  - TSV 户型表导入后，公寓详情户型可读取。
