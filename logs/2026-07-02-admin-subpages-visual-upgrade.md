# 2026-07-02 管理员子页面视觉升级

## 改动概述

- 优化管理员入口下 7 个子页面：公寓管理、户型管理、活动管理、服务管理、物品管理、评论管理、用户管理。
- 将通用 CRUD 页面升级为更贴近设计稿的后台工具界面：
  - 新增页面说明区，展示模块名称、说明和业务徽标。
  - 优化搜索、新增、筛选和统计区域的层级与间距。
  - 记录卡片增加业务标签、字段分组和类型化状态。
  - 不同模块使用不同摘要标题、状态文案和操作按钮。
  - 表单弹窗改为底部抽屉式结构，提升移动端编辑体验。
- 扩展静态管理数据字段：
  - 活动增加人数限制、发起人。
  - 服务增加预约时间、接单人。
  - 物品增加借用规则。
  - 用户增加身份字段。
- 保持原生小程序实现，不引入第三方库，不接真实接口。
