# 2026-07-20 XLSX 文件导入导出闭环

## 已实现流程

1. 管理员点击现有“批量导出”。
2. 云函数将当前表格数据生成真实 `.xlsx` 文件并写入云存储。
3. 小程序下载后直接调用微信原生文档页；管理员在右上角菜单保存文件。
4. 管理员用 Excel/WPS 修改保存后的 `.xlsx`。
5. 点击现有“批量导入”，选择 `.xlsx`；小程序上传文件至云存储。
6. 云函数解析第一个工作表，转换为既有 CSV 校验管线的文本；随后沿用预览、校验和确认导入流程。

## 兼容性

- 公寓导出文件名：`公寓导出.xlsx`。
- 户型导出文件名：`户型导出.xlsx`。
- CSV/TXT 文件导入保持兼容。
- 不再显示“转发到微信”或 CSV 打开操作弹窗。

## 上线前必须操作

1. 在微信开发者工具右键 `cloudfunctions/rencai`，选择“上传并部署：云端安装依赖”。
2. 由于云函数新增 `xlsx` 依赖，未部署最新云函数时，线上仍会使用旧的 CSV 生成/导入行为。
3. 使用真机完成验收：导出 → 微信文档页保存 → 用 Excel/WPS 修改 → 批量导入选择该 `.xlsx` → 预览 → 确认。

## 已验证

- 导出 XLSX buffer 可由 Excel 解析库读回，且保留原始中文表头。
- XLSX 首工作表可转换为现有 CSV 导入管线文本。
- 管理页测试覆盖：下载后打开 XLSX 系统保存页、移除转发入口、上传 XLSX 后创建任务并跳转预览。
- 完整 Node 测试、JavaScript 语法检查和差异格式检查在交付前执行。
