# 公寓费用标签统一设计

## 目标

将公寓费用中的“网费”统一为“暖气费”，并让详情页、Excel 导入和 Excel 导出使用同一套六项费用定义。

## 规则

- 详情页固定显示：水费（m³）、电费（度）、燃气费（m³）、物业费（㎡/月）、暖气费（㎡/天）、停车费（月）。
- 纯数字显示为“数字元”；文字如“无”“已含”原样显示。
- 新 Excel 表头使用水费（元/m³）、电费（元/度）、燃气费（元/m³）、物业费（元/㎡/月）、暖气费（元/㎡/天）、停车费（元/月）。
- 旧“网费（元/月）”表头不兼容，导入必须失败并提示使用新版模板。
- 云数据库所有已有“网费”费用项更名为“暖气费”，值、状态和顺序不变。

## 验证

测试数字/文字显示、严格表头识别、导入导出表头；迁移前备份，迁移后回读所有记录并核对无“网费”。
