# 真机 CSV 文件导出设计

## 目标

保留管理员页面现有的“批量导出”按钮。管理员在真机微信点击后，获得可打开的 CSV 文件，并可转发给文件传输助手或保存；不新增页面、按钮或下载链接入口。

## 范围

- 支持公寓与户型的既有 CSV 字段、筛选和导出命名。
- 保持封面路径及公寓平面图 JSON 的既有导出规则。
- 移除“分享失败则复制 CSV 文本”的成功降级行为。

## 数据流

1. 现有页面从云函数读取管理员可导出的公寓或户型数据，并在前端生成 UTF-8 BOM CSV 内容。
2. 页面调用新增的受 `requireAdmin` 保护的云函数 action，将文件名和 CSV 内容提交给云函数。
3. 云函数将内容写入临时文件，上传至云存储 `exports/`，并返回 `fileID`。
4. 真机小程序用 `wx.cloud.downloadFile` 下载 `fileID`，再以 `wx.openDocument({ showMenu: true })` 打开 CSV。
5. 用户通过文件预览的系统菜单转发或保存文件。

## 失败处理

- 云端生成失败：提示“导出文件生成失败，请重试”。
- 文件下载失败：提示“导出文件下载失败，请重试”。
- 文件预览失败：提示“文件已生成，请在微信文件中查看”。
- 不复制 CSV 到剪贴板，也不显示“导出成功”但没有文件的提示。

## 安全与边界

- 新 action 仅允许管理员调用，且只接受 `apartments` / `room_types` 对应的导出文件名。
- CSV 内容限制在合理大小，防止滥用云函数存储；文件保存在受云环境权限控制的云存储中。
- 不改变现有数据查询和筛选权限；真实数据仍由现有 `exportAdminItems` 白名单读取。

## 验收

- 公寓与户型的现有“批量导出”按钮保持唯一导出入口。
- 真机点击后出现可预览的 `.csv` 文件，预览菜单可转发。
- 生成、下载、预览失败均有准确提示，且不再复制完整 CSV 文本。
- 新增单元测试覆盖云函数返回文件 ID 与页面下载/打开调用。
